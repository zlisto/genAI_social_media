Setup Instructions

VS Code

  • Download and install VS Code from the official website
  • Install Python from python.org
  • Install Node.js and npm to enable React development
  • Install the Python extension in VS Code
  • Install the JavaScript/TypeScript and React extensions
  • Set up terminal and interpreter paths correctly
  • Enable Git integration if desired
  • Install GitHub Copilot if you have a ChatGPT Pro account

Cursor

  • Download Cursor from the official site (cursor.sh)
  • Register with your .edu email for free student access
  • Follow install instructions based on your OS
  • Sign in and connect GitHub/ChatGPT if prompted

ChatGPT

  • Go to openai.com and create an account
  • Verify your email and phone number
  • Upgrade to ChatGPT Plus if you want access to GPT-4 and tools

ElevenLabs

  • Go to elevenlabs.io
  • Create a free account
  • Verify your email address
  • Get your API key from the dashboard (needed for assignments)
