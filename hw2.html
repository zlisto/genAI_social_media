Homework 2: Persona Reaction Generator

Instructions

  1. Create a Streamlit web app that allows a user to:

    1. Home page: Input a name and text about the persona in two input boxes and then click a button to create the persona. This should save the persona in the app.
    2. Reaction page: Choose one of your personas from a dropdown menu. Then upload an image and click a button to have the persona react to it, then show the reaction text in a box below. Make sure the image is displayed in the app with width 400.

  2. Your app should either import the GenAI class from the genai.py file that is provided: genai.py or build your backend in n8n.

  4. The app should look clean and be easy to use.

  5. Your submission should include the following files:

    1. app.py - the main app file
    2. If you build your backend in n8n, you should include the n8n workflow file (JSON file).
    3. If you build your backend in python, include any additional python files you used.

Grading Breakdown (100 Points)

Component Points Description
App Functionality 10 App runs without crashes or major bugs; all pages are accessible
Persona Creation (Home Page) 15 User can input a name and text description to define a persona and save it
Persona Storage 10 Created personas are stored correctly and accessible from the dropdown
Reaction Page – Image Upload 10 User can upload an image and see it displayed at width 400
Reaction Page – Persona Output 15 Persona reacts to the uploaded image and the reaction is clearly shown
Backend Integration 15 App uses either the provided GenAI class or a working n8n backend
Interface and Design 10 App layout is clean, intuitive, and visually organized
Code Quality 10 Code is readable, modular, and well-documented; uses appropriate helper functions
Submission Completeness 5 All required files (e.g. app.py, GenAI/n8n workflow, others) are submitted properly
