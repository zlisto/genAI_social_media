Homework 1: Text-to-Image Instagram Caption Generator

Instructions

Create a Streamlit web app that allows a user to:

  • Upload an image (e.g., a travel photo, product photo, selfie, etc.)
  • Enter a short text description of the content creator (e.g., their niche, tone, interests)
  • Use an AI model to generate an Instagram caption based on both the image and the creator's info
  • Display the final caption underneath the image

You may use OpenAI or any other image-to-text or captioning model. The app should look clean and easy to use.

Grading Breakdown (40 Points)

Component Points Description
App Functionality 10 Streamlit app runs smoothly without crashes; layout is clean and readable
Image Upload 10 User can upload and preview image, which is processed by the app
Creator Context Input 10 Textbox for influencer background/niche/tone; info passed to AI model
AI Captioning 10 Caption clearly uses both image content and user-provided creator info
