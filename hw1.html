Homework 1: Fashion Article Generator

Instructions

  1. Create a Streamlit web app that allows a user to:

    1. Home page: Upload multiple images
    2. Article page: Enter a short text description of the article topic (e.g., author's tone, speaking style, theme of photos, etc) and use AI to generate a nicely formatted HTML blog post based on both the images and the text description
    3. Items page: Select from the uploaded images and use AI to generate a nicely formatted HTML table of the name, description, and estimated price of the items in the images

  2. Your app should import the GenAI class from the genai.py file that is provided: genai.py.

  3. The user should only have to upload the images once on the homepage and use the same images for both the article and items pages.

  4. The app should look clean and be easy to use.

  5. Your submission should include the following files:

    1. app.py - the main app file
    2. utils.py - a file that contains helper functions for the app
    3. genai.py - a file that contains the GenAI class

Grading Breakdown (100 Points)

Component Points Description
App Functionality 10 Streamlit app runs smoothly without errors or crashes
Image Upload (Home Page) 15 User can upload and preview multiple images on the homepage
State Persistence 10 Uploaded images persist across all pages using session state
Article Page – Input 10 Text input for style description is clearly presented
Article Page – Output 15 Blog article is generated using uploaded images and user input; HTML is properly displayed
Items Page – Image Selection 10 User can select one image from the uploaded set to analyze on the Items page
Items Page – Table Output 10 HTML table of item descriptions and prices is generated and displayed correctly
Code Structure 10 App uses helper functions from utils.py for blog writing and item table generation
Visual Presentation 10 Interface is clean, organized, and easy to navigate across pages
